## Overview

Teacher Diary is an Apple-native application designed to support **teacher wellbeing, student behaviour tracking, and professional reflection** in a real classroom environment.

The app prioritises **speed, privacy, and daily usability**, making it suitable for iPad and Mac use in schools.

---

## Educational Context

- Target users: Classroom teachers
- Context: Daily teaching, behaviour logging, reflection, wellbeing tracking
- Environment: iPad and macOS (Apple-centric schools)

This app was designed from lived classroom experience, not as a theoretical prototype.

---

## Core Features

- Student behaviour and observation logging
- Reflection and note-taking workflows
- Date-based and student-based organisation
- Local data storage (privacy-first)
- Fast, minimal-friction UI for in-class use

---

## AI Readiness (Planned / Extensible)

This project is architected to support **responsible AI enhancement**, including:

- On-device pattern detection using Core ML
- Reflection prompt suggestions (teacher-controlled)
- Behaviour trend insights without student profiling
- Optional teacher-only AI summaries

Importantly:
- No student data is sent to third-party AI services
- AI is advisory, not decision-making
- Teachers remain in control at all times

---

## Safety & Privacy Principles

- No student accounts required
- All data stored locally on device
- No background analytics or tracking
- Designed to align with school privacy expectations
